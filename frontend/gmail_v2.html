Hướng Dẫn Cài Đặt Developer
Admin cần thực hiện 1 lần để kích hoạt Gmail Integration
1
Tạo Google Cloud Project
Vào console.cloud.google.com → New Project → Đặt tên "SmartFinance"
2
Bật Gmail API
APIs & Services → Enable APIs → Tìm "Gmail API" → Enable
3
Tạo OAuth 2.0 Credentials
APIs & Services → Credentials → Create Credentials → OAuth Client ID
Application type: Web application
Authorized redirect URI: https://smart-finance-lbei.onrender.com/api/gmail/callback
4
Thêm vào Render Environment Variables
Copy Client ID và Client Secret → Thêm vào Render:
GOOGLE_CLIENT_ID = your-client-id.apps.googleusercontent.com
GOOGLE_CLIENT_SECRET = your-secret
GOOGLE_REDIRECT_URI = https://smart-finance-lbei.onrender.com/api/gmail/callback
5
OAuth Consent Screen
APIs & Services → OAuth consent screen → External → Điền thông tin app
Scopes: thêm gmail.readonly
Test users: thêm email của bạn vào danh sách
